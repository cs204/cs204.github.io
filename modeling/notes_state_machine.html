Автоматы

Автомат - это метод моделирования системы вывод которой зависит от истории ввода. Мы можем задать автомат указав:

  • множество состояний, S
  • множество вводов, I
  • множество выводов, O
  • функцию следующего состояния, \(n(i_t, s_t) \mapsto s_{t+1} \), для ввода и состояния в момент \(t \) задаёт состояние в момент \(t+1\)
  • функцию вывода, \( o(i_t, s_t) \mapsto o_t \), для ввода и состояния в момент \(t\) задаёт вывод в момент \(t\); и
  • начальное состояние, \(s_o\), состояние в момент времени 0.

Автомат в Python


class SM:
    def start(self):
        self.state = self.startState

    def step(self, inp):
        (s, o) = self.getNextValues(self.state, inp)
        self.state = s
        return o

    def transduce(self, inputs, verbose = False):
        self.start()
	return [self.step(inp) for inp in inputs]

Автомат с задержкой


class Delay(SM):
    def __init__(self, v0):
        self.startState = v0

    def getNextValues(self, state, inp):
        return (inp, state)

Incrementor


class Increment(SM):
    def __init__(self, incr):
        self.incr = incr

    def getNextState(self, state, inp):
        return safeAdd(inp, self.incr)
Вспомогательная функция safeAdd, понадобится для соединения с задержкой. 

def safeAdd(a, b):
    if a == 'undefined' or b == 'undefined':
        return 'undefined'
    else:
        return a + b

Каскадное соединение


class Cascade(SM):
    def __init__(self, sm1, sm2):
        self.m1 = sm1
        self.m2 = sm2
        self.startState = (sm1.startState, sm2.startState)

    def getNextValues(self, state, inp):
        (s1, s2) = state
        (newS1, o1) = self.m1.getNextValues(s1, inp)
        (newS2, o2) = self.m2.getNextValues(s2, o1)
        return ((newS1, newS2), o2)

Параллельное соединение


class Parallel(SM):
    def __init__(self, sm1, sm2):
        self.m1 = sm1
        self.m2 = sm2
        self.startState = (sm1.startState, sm2.startState)

    def getNextValues(self, state, inp):
        (s1, s2) = state
        (newS1, o1) = self.m1.getNextValues(s1, inp)
        (newS2, o2) = self.m2.getNextValues(s2, inp)
        return ((newS1, newS2), (o1, o2))

        (newS2, o2) = self.m2.getNextValues(s2, i2)
        return ((newS1, newS2), (o1, o2))

Автомат обратной связи


class Feedback(SM):
    def __init__(self, sm):
        self.m = sm
        self.startState = self.m.startState

    def getNextValues(self, state, inp):
        (ignore, o) = self.m.getNextValues(state, 'undefined')
        (newS, ignore) = self.m.getNextValues(state, o)
        return (newS, o)



class Feedback2(Feedback):
    def getNextValues(self, state, inp):
        (ignore, o) = self.m.getNextValues(state, 'undefined')
        (newS, ignore) = self.m.getNextValues(state, (inp, o))
        return (newS, o)
