Автоматы

Автомат - это метод моделирования системы, вывод которой зависит от истории ввода.

Мы можем задать автомат указав:

  • множество состояний, S
  • множество вводов, I
  • множество выводов, O
  • функцию следующего состояния, \(n(i_t, s_t) \mapsto s_{t+1} \), для ввода и состояния в момент \(t \) задаёт состояние в момент \(t+1\)
  • функцию вывода, \( o(i_t, s_t) \mapsto o_t \), для ввода и состояния в момент \(t\) задаёт вывод в момент \(t\); и
  • начальное состояние, \(s_o\), состояние в момент времени 0.

Автомат задержки

Автомат берет ввод и отправляет его на вывод, но с задержкой на один шаг.

$$\begin{array}{ccc} S &=& anything\\ I &=& anything\\ O &=& anything\\ n(s, i) &=& i\\ o(s, i) &=& s\\ s_0 &=& 0\\ \end{array} $$

Дана последовательность \(i_0, i_1, i_2, ..\), автомат будет выводить \(0, i_0, i_1, i_2, ..\).

Аккумулятор

Вывод из аккумулятора равен сумме всех вводов.

$$\begin{array}{ccc} S &=& numbers\\ I &=& numbers \\ O &=& numbers \\ n(s,i) &=& s + i\\ o(s, i) &=& n(s, i)\\ s_0 &=& 0 \end{array} $$
time 0 1 2 3
input 100 -3 4 -123
state 0 100 97 101
output 100 97 101 -22

Автомат принимающий язык

Рассмотрим автомат, вывод из него True, если вводимая строка удовлетворяет шаблону, False в противном случае. Пусть шаблон будет a, b, c, a, b, c, a, b, c, ... Автомат использует состояния 0, 1, 2, в которых он ожидает a, b, c соответственно, и он использует состояние 3 для случаев, когда он встретил не то, что ожидал. Когда автомат попадает в состояние 3, он ни когда его не покидает.

$$ \begin{array}{ccc} S &=& \{0, 1, 2, 3\}\\ I &=& \{a, b, c\} \\ O &=& \{True, False\}\\ s_0 &=& 0 \end{array} $$ $$ n(s,i) = \left\{ \begin{array}{ccc} 1, &if& s=0, i=a\\ 2, &if& s=1, i=b\\ 0, &if& s=2, i=c\\ 3, &&otherwise&\\ \end{array}\right. $$ $$ o(s, i) = \left\{ \begin{array}{ccc} False, &if& n(s,i)=3\\ True && otherwise\\ \end{array}\right. $$

Автомат в Python

Зададим абстрактный класс SM, он будет родителем для остальных классов автоматов.


class SM:
    def start(self):
        self.state = self.startState

    def step(self, inp):
        (s, o) = self.getNextValues(self.state, inp)
        self.state = s
        return o

    def transduce(self, inputs, verbose = False):
        self.start() 
        return [self.step(inp) for inp in inputs]

Метод getNextValues берет состояние и ввод в момент t и возвращает состояние в момент t+1 и вывод в момент t. Метод step берет ввод, изменяет состояние автомата и возвращает вывод. Метод transduce для последовательности вводов возвращает выводы.

Автомат с задержкой


class Delay(SM):
    def __init__(self, v0):
        self.startState = v0

    def getNextValues(self, state, inp):
        return (inp, state)

Incrementor

Автомат Incrementor берёт ввод и возвращает его плюс константа incr. 


class Increment(SM):
    def __init__(self, incr):
        self.incr = incr

    def getNextState(self, state, inp):
        return safeAdd(inp, self.incr)
Вспомогательная функция safeAdd, понадобится для соединения с задержкой. 

def safeAdd(a, b):
    if a == 'undefined' or b == 'undefined':
        return 'undefined'
    else:
        return a + b

Каскадное соединение

Две машины соединяем каскадом. Вывод из первой машины направляем на вход второй. 


class Cascade(SM):
    def __init__(self, sm1, sm2):
        self.m1 = sm1
        self.m2 = sm2
        self.startState = (sm1.startState, sm2.startState)

    def getNextValues(self, state, inp):
        (s1, s2) = state
        (newS1, o1) = self.m1.getNextValues(s1, inp)
        (newS2, o2) = self.m2.getNextValues(s2, o1)
        return ((newS1, newS2), o2)

Параллельное соединение


class Parallel(SM):
    def __init__(self, sm1, sm2):
        self.m1 = sm1
        self.m2 = sm2
        self.startState = (sm1.startState, sm2.startState)

    def getNextValues(self, state, inp):
        (s1, s2) = state
        (newS1, o1) = self.m1.getNextValues(s1, inp)
        (newS2, o2) = self.m2.getNextValues(s2, inp)
        return ((newS1, newS2), (o1, o2))

Автомат обратной связи


class Feedback(SM):
    def __init__(self, sm):
        self.m = sm
        self.startState = self.m.startState

    def getNextValues(self, state, inp):
        (ignore, o) = self.m.getNextValues(state, 'undefined')
        (newS, ignore) = self.m.getNextValues(state, o)
        return (newS, o)



class Feedback2(Feedback):
    def getNextValues(self, state, inp):
        (ignore, o) = self.m.getNextValues(state, 'undefined')
        (newS, ignore) = self.m.getNextValues(state, (inp, o))
        return (newS, o)

Управление устройством

Мы хотим управлять устройством (plant). Выход из устройства (обычно это показания датчиков) отправляем на вход в контроллер и выход из контроллера (обычно это действия, управляющие устройством) отправляем на вход устройства.

Рассмотрим робота, двигающегося к стене. Он имеет сенсор определяющий расстояние до стены \(d[t]\). Мы хотим, чтобы робот остановился на заданном расстоянии от стены (\( d_{desired} \)). На вход робота подаётся скорость \(v[t]\), с которой он должен двигаться. Выход из контроллера:

$$v[t]=k(d_{desired}-d[t]).$$

Выход из робота:

$$ d[t]=d[t-1]+V[t-1]\delta T, $$

где \(\delta T\) - время между командами задания скорости. Реализуем эту систему на python: 


import sm
k = 1.5
dDesired = 1.0
class WallContorller(sm.SM):
    def getNextState(self, state, inp):
        return sm.safeMul(k, sm.safeAdd(dDesired, sm.safeMul(-1, inp)))

deltaT = 0.1
class WallWorld(sm.SM):
    startState = 5
    def getNextValues(self, state, inp):
        return (sm.safeAdd(state, sm.safeMul(inp, deltaT)),
                state)

def coupledMachine(m1, m2):
    return sm.Feedback(sm.Cascade(m1, m2))

wallSim = coupledMachine(WallContorller(), WallWorld())
print(wallSim.run(30))
