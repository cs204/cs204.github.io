Open source workflow.

Зайдите на https://github.com/cs203/413. Нажмите на кнопку Fork. У вас будет создана копия репозитория. Скопируйте его адрес. Зайдите в cs50.dev. Отключите плагин gitdoc. Перейдите в домашний каталог, сколонируйте репозиторий. 


cd ~
git clone адрес
Перейдите в каталог 413. Создайте ветку dev, перейдите в неё. 

cd 413
git branch dev
git checkout dev
Создайте свою веб страницу на любую тему, сделайте на неё ссылку из index.html. Выполните 

git add .
git commit -m "ваше название"
Добавьте информацию об основном репозитории. Перейдите в ветку main. 

git remote add upstream https://github.com/cs203/413.git 
git checkout main
Закачайте изменения. Соедините их с вашими. Загрузите результат. 

git fetch upstream
git merge upstream
git merge dev
git push origin main
Зайдите на github, создайте pull request.
