Турникет

Ниже приведена диаграмма переходов состояний для турникета.

В файле turnstile.py завершите определение класса Turnstile, замените Q1, Q2, Q3, Q4, Q5, Q6,...


import sm
class Turnstile(sm.SM):
	startState = 'locked'
	def getNextValues(self, state, inp):
		if state == 'locked':
			if inp == 'coin':
				return (Q1, Q2)
			else:
				return (Q3, Q4)
		else:	# state == 'unlocked'
			if inp == 'turn':
				return (Q5, Q6)
			else:
				return (Q7, Q8)
Используйте класс SM из файла sm.py.

Шаги выполнения

Зайдите на cs50.dev. Сделайте рабочим каталогом вашу Codespace. Создайте каталог turnstile, выполнив 
mkdir turnstile
Перейдите в каталог. 
cd turnstile
Закачайте файлы sm.py, turnstile.py выполнив 

wget https://vadimgb.github.io/psets/data/sm.py 
wget https://vadimgb.github.io/psets/data/turnstile.py
Откройте файл turnstile.py, выполнив 

code turnstile.py
Дополните файл.

Как проверить

  • Сделайте рабочим каталог с программой,
  • Запустите программу, выполнив python turnstile.py.
Вы можете проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/turnstile local
Для отправки на проверку выполните команду 
npx cs204 2023/turnstile
