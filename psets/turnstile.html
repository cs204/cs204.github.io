Турникет

Ниже приведена диаграмма переходов состояний для турникета.

В файле trunstile.py завершите определение класса Turnstile, замените Q1, Q2, Q3, Q4, Q5, Q6,...


class Turnstile1(SM):
	startState = 'locked'
	def getNextValues(self, state, inp):
		if state == 'locked':
			if inp == 'coin':
				return (Q1, Q2)
			else:
				return (Q3, Q4)
		else:	# state == 'unlocked'
			if inp == 'turn':
				return (Q5, Q6)
			else:
				return (Q7, Q8)
Используйте класс SM из файла sm.py.
