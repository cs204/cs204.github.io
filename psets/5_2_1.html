Функция системы поиска стен

Используйте sf.R, sf.Gain, sf.Cascade, sf.FeedbackAdd и sf.FeedbackSubtract для создания системной функции робота-искателя стен.

Эти процедуры описаны в модуле sf документации по программному обеспечению.

Определите процедуру python под названием controllerSF, которая принимает один аргумент:

  • пропорциональный коэффициент k, применяемый к ошибке.

и которая возвращает системную функцию системы, входными данными которой является ошибка расстояния, а выходными данными — заданная скорость.

Определите процедуру Python под названием plantSF, которая принимает один аргумент:

  • продолжительность временного шага T.

и которая возвращает системную функцию для системы, входными данными которой является заданная «скорость», а выходными данными — фактическое расстояние до стены.

Определите процедуру python под названием sensorSF, которая не принимает аргументов и возвращает системную функцию для системы, входные данные которой — фактическое значение датчика, а выходные данные — показания датчика с задержкой на один шаг.

Определите процедуру python под названием wallFinderSystemSF, которая принимает два аргумента:

  • шаг времени T.
  • коэффициент пропорциональности k, применяемый к ошибке

и которая возвращает системную функцию для системы, входными данными которой является желаемое расстояние, а выходными данными — фактическое расстояние до стены.

Вы можете отладить их, поместив свои определения в файл и включив import lib601.sf as sf вначале файла.

Вы можете проверить, получили ли вы правильный ответ, сравнив полученную системную функцию с разностными уравнениями для различных систем, которые вы получили в задачах предыдущей лабораторной работы по проектированию. Для сравнения вам нужно будет преобразовать разностные уравнения в системные функции (или наоборот) — будьте внимательны к порядкам коэффициентов и знаков минус...
