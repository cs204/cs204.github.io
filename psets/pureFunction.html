Шаги выполнения

Зайдите на cs50.dev. Сделайте рабочим каталогом вашу Codespace. Создайте каталог pureFunction, выполнив 
mkdir pureFunction
Перейдите в каталог. 
cd pureFunction
Закачайте файлы sm.py, pureFunction.py выполнив 

wget https://vadimgb.github.io/psets/data/sm.py 
wget https://vadimgb.github.io/psets/data/pureFunction.py
Откройте файл pureFucntion.py, выполнив 

code pureFunction.py
Дополните файл. Вы можете проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/pureFunction local
Для отправки на проверку выполните команду 
npx cs204 2023/pureFunction
