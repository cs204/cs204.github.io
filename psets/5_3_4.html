Модель следующего за препятствием

Используйте sf.R, sf.Gain, sf.Cascade, sf.FeedbackAdd и sf.FeedbackSubtract для построения функции роботизированной системы поиска стен. Эти процедуры описаны в модуле sf документации по программному обеспечению.

Определите процедуру Python под названием wallFollowerModel, которая принимает три аргумента:

  • пропорциональный коэффициент k, применяемый к ошибке.
  • длительность временного шага T
  • поступательная скорость V

и которая возвращает системную функцию для системы, входные данные которой — желаемое расстояние, а выходные данные — фактическое расстояние до стены.

Их можно отладить в файле designLab05Work.py.
