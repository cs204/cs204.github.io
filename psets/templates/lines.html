Строки кода

Один из способов оценить сложность программы - посчитать количество строк кода, исключиая пустые строки и комментарии. Например программа hello.py: 
    
    #пробный файл для lines.py

    name = input("Waht's you name? ")
    print(f"hello, {name}")
имеет всего две строки кода.

В файле lines.py реализуйте программу, которая ожидает один аргумент командной строки, имя файла python, возвращает число строк, исключая пустые строки и комментарии. Если пользователь не указывает точно один аргумент командной строки, или файл имеет расширение не py, или файла с указанным именем нет, программа заканчивает работу через вызов sys.exit.

Предполагаем, что срока комментария начинается с #. Создайте файл hello.py: 

    
    #пробный файл для lines.py

    name = input("Waht's you name? ")
    print(f"hello, {name}")
Он нужен для проверки.

Подсказка
  • Тип str имеет много методов, см https://digitology.tech/docs/python_3/library/stdtypes.html#string-methods, включая lstrip и startswith.
  • Функция open бросает исключение FileNotFoundError, см https://digitology.tech/docs/python_3/library/exceptions.html#FileNotFoundError.

Демонстрация

Шаги выполнения

Зайдите на cs50.dev. Используя команду cd, сделайте рабочим каталогом codespace. Создайте каталог lines, выполнив 
mkdir lines
Перейдите в каталог. 
cd lines
Выполните 

code lines.py
чтобы открыть редактор, и напишите вашу программу. Создайте файл hello.py с текстом, приведенным выше.

Как проверить

Сделайте рабочим каталог с программой.

  • Запустите программу python lines.py. Ваша программа должна завершить работу с sys.exit и вывести сообщение: 
    
Слишком мало аргументов в командной строке
  • Создайте две программы hello.py и goodbye.py. Выполните в терминале команду python lines.py hello.py goodbye.py. Ваша программа должна завершить работу с sys.exit, выести сообщение: 
    
Слишком много аргументов в командной строке
  • Создайте файл invalid_extension.txt. Выполните python lines.py invalid_extension.txt. Программа должна завершить через sys.exit, вывести: 
    
Не python файл
  • Запустите программу python non_existen_file.py. Предполагается, что такого файла нет в папке, программа должна обработать исключение и завершить работу через sys.exit, вывести сообщение: 
        
    Файл не существует
  • Создайте файл hello.py. Как указано ниже. 
        
    #пробный файл для lines.py

    name = input("Waht's you name? ")
    print(f"hello, {name}")
    Запустите python lines.py hello.py. Программа должна вывести: 
            
        2
