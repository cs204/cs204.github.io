Open source workflow.

Зайдите на https://github.com/cs203/413. Нажмите на кнопку Fork. У вас будет создана копия репозитория. Скопируйте его адрес, перейдите в домашний каталог, сколонируйте его. 


cd ~
git clone адрес
Отключите плагин gitdoc. Перейдите в каталог 413. Создайте свою веб страницу на любую тему, сделайте на нее ссылку из index.html. Выполните 

git add .
git commit -m "ваше название"
Добавьте информацию об основном репозитории 

git remote add upstream https://github.com/cs203/413.git
Закачайте изменения. Соедините их с вашими. Загрузите результат. 

git fetch upstream
git merge upstream
git push origin ветка
Зайдите на github, создайте pull request.
