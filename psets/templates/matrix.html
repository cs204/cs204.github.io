Matrix.

Используйте 24.6 из Programming principles and practice using C++ (2014). Используя 

Vector classical_gaussian_elimination(Matrix A, Vector b)
для решения системы


1x+4y+3z=17, 3x+2y+1z=13, 4x+2y+3z=17;

Шаги выполнения

  • Зайдите на https://cs50.dev.
  • Загрузите дистрибутив. 
    		 
		wget https://cs204.github.io/psets/data/matrix.zip
  • Распакуйте. 
    		
		unzip matrix.zip
  • Сделайте папку matrix рабочей. 
    		
		cd matrix
  • Завершите определение функции.

Как проверить

Сделайте рабочим каталог с программой.

Вы можете проверить правильность выполнения задания, выполнив команду 
npx cs204 cpp/matrix local
Для отправки на проверку выполните команду 
npx cs204 cpp/matrix
