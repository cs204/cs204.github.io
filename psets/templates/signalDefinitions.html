Задание сигнала

Закачайте файл:


wget https://vadimgb.github.io/psets/data/signalDefinitions.py

Реализуйте свою версию сигналов.

  • StepSignal(): имеет значение 0 для аргумента меньшего 0 иначе 1.
  • SummedSignal(s1, s2): Принимает два сигнала, s1 и s2, во время инициализации и создаёт новый сигнал, который представляет собой сумму этих сигналов. Обратите внимание, что этот класс должен быть ленивым: при создании сигнала не должно происходить никаких сложений; сложение должно выполняться только когда вызывается sample SummedSignal.
  • ScaledSignal(s,c): принимает сигнал s и константу c во время инициализации и создает новый сигнал, значения sample которого представляют собой значения sample исходного сигнала, умноженные на c.
  • R(s): Принимает сигнал s во время инициализации и создаёт новый сигнал, sample которого задерживаются на один временной шаг; то есть значение нового сигнала в момент времени n должно быть значением старого сигнала в момент времени n - 1.
  • Rn(s, n): принимает сигнал s во время инициализации и создаёт новый сигнал, sample которого задерживаются на n шагов.

Для проверки раскомментируйте строки в конце файла.

Вы можете проверить правильность выполнения задания, выполнив команду 

npx cs204 2023/signalDefinitions local
Для отправки на проверку выполните команду 
npx cs204 2023/signalDefinitions
