ide. Задание.

Создайте пару ключей.

Зайдите в cs50.dev. Выполните в терминале 


ssh-keygen
cat ~/.ssh/id_rsa.pub
Скопируйте распечатанный публичный ключ. Перейдите на https://github.com/settings/keys, создайте новый ssh ключ, в поле key вставьте скопированный публичный ключ, нажмите add ssh key.

Регистрация на сайте учета заданий

Используя учётную запись на github, зайдите на сайт http://90.188.117.161:8080. Вам будет отправлено письмо с темой cs203. В веб-браузере устройства, на котором будете смотреть письмо, надо войти на github. Примите находящееся в письме приглашение (View Invitation -> Accept Invitation). Письмо может попасть в спам. Для вас был создан репозиторий, вы будете отправлять в него свой работы на проверку. Имя репозитория имеет вид https://github.com/cs204/{username}.git, где {username} замените на имя пользователя на github. Так если пользователь имеет имя в github vadimgb1, то репозиторий будет называться https://github.com/cs204/vadimgb1.git, где cs204 - это имя организации.

Выполнение первого задания

Зайдите на cs50.dev. Создайте каталог и перейдите в него. 
mkdir hello
cd hello
Создайте файл hello.cpp 

code hello.cpp
Наберите в нем текст 

#include <iostream>

class World
{
	public:
		World(){ std::cout << "Hello!\n";}
		~World(){std::cout << "Good bye!\n";}
};

World TheWorld;

int main()
{}
Выполните команду в терминале 
g++ -o hello hello.cpp
Компилятор c++ создаст выполняемый файл. Выполните созданную вами программу. 
./hello
Чтобы проверить правильность выполнения задания, выполните команду 
npx cs204 cpp/hello local
Для отправки на проверку выполните команду 
npx cs204 cpp/hello
