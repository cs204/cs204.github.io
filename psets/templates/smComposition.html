Композиция автоматов

Используйте sm.R, sm.Gain, sm.Cascade, sm.FeedbackAdd and sm.FeedbackSubtract чтобы задать следующие автоматы. Это те системы, которые были в предыдущем задании.

Задайте процедуру accumulator, она беерт один ввод:

  • вывод до первого ввода, init.
и возвращмет автомат, вывод которого в момент n равен сумме вводов до момента n включительно. Таким образом, выход в момент 0 представляет собой сумму init и ввода в момент 0.

Задайте процедуру accumulatorDelay, которая берет один аргумент:

  • вывод в момент времени 0, init.
Возвращает автомат вывод из которого в момент n равен сумме вводов до момента n-1 включительно.

Задайте процедуру accumulatorDelayScaled, которая берет два аргумента:

  • костанту умножения.
  • вывод в момент 0, init.
Возвращает автомат вывод из которого в момент n равен сумме умноженных на костанту вводов до момента n-1 включительно.
