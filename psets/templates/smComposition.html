Композиция автоматов

Используйте sm.R, sm.Gain, sm.Cascade, sm.FeedbackAdd and sm.FeedbackSubtract, чтобы задать следующие автоматы. Это те системы, которые были в предыдущем задании.

Задайте процедуру accumulator, она берёт один ввод:

  • вывод до первого ввода, init
и возвращает автомат, вывод которого в момент n равен сумме вводов до момента n включительно. Таким образом, выход в момент 0 представляет собой сумму init и ввода в момент 0.

Задайте процедуру accumulatorDelay, которая берет один аргумент:

  • вывод в момент времени 0, init.
Возвращает автомат вывод из которого в момент n равен сумме вводов до момента n-1 включительно.

Задайте процедуру accumulatorDelayScaled, которая берет два аргумента:

  • константу умножения,
  • вывод в момент 0, init,
возвращает автомат, вывод из которого в момент n равен сумме умноженных на константу вводов до момента n-1 включительно.

Шаги выполнения

  • Скачайте архив: 
    		
		wget https://vadimgb.github.io/psets/data/smComposition.zip
  • Распакуйте, выполнив: 
    		
		unzip smComposition.zip
  • Сделайте рабочим каталог smComposition: 
    		
		cd smComposition
  • Откройте файл: 
    		
		code smComposition.py
  • Задайте нужные функции.
  • Раскомментируйте строки в функции test.

Проверка

Вы можете проверить правильность выполнения задания, выполнив команду: 
npx cs204 2023/smComposition local
Для отправки на проверку выполните команду: 
npx cs204 2023/smComposition
