Автомат комментариев

Создайте автомат, на его ввод посылаем символы строки (представляющей программу на Python), вывод из автомата или (а) символы, если это часть комментария или (б) None. Как вы знаете, комментарии начинаются с '#' и продолжаются до конца текущей строки.

Наример:


>>str='''def f(x): #comment
	return 1'''
>>m = CommentsSM()
>>m.transduce(str)
[None, None, None, None, None, None, None, None, None, None,
'#', ' ', 'c', 'o', 'm', 'm', 'e', 'n', 't',
None, None, None, None, None, None, None, None, None, None, None, None]

Символ конца строки '\n' вы можете проверить:


if inp == '\n'

Начните с диаграммы состояний, отображающей состояния и какие вводы в какие состояния переводят автомат. Закачайте файлы comments.py, sm.py. Откройте файл comments.py, дополните определение класса.

Шаги выполнения

Зайдите на cs50.dev. Сделайте рабочим каталогом вашу Codespace. Создайте каталог comments, выполнив 
mkdir comments
Перейдите в каталог. 
cd comments
Закачайте файлы sm.py, comments.py выполнив 

wget https://cs204.github.io/psets/data/sm.py 
wget https://cs204.github.io/psets/data/comments.py
Откройте файл comments.py, выполнив 

code comments.py
Дополните файл.

Как проверить

  • Сделайте рабочим каталог с программой,
  • Запустите программу, выполнив python comments.py.
Вы можете проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/comments local
Для отправки на проверку выполните команду 
npx cs204 2023/comments
