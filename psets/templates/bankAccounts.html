Объединение счетов

Рассмотрим два разных типа банковских счетов:

  • B1: комиссия в размере 100$ за каждый (ненулевой) депозит и вывод средств; 2% годовых за каждый временной шаг. 
    			
			class BA1(sm.SM):
			    startState = 0
			    def getNextValues(self, state, inp):
				if inp != 0:
				    newState = state * 1.02 + inp - 100
				else:
				    newState = state * 1.02
				return (newState, newState)
  • B2: отсутствие комиссии за транзакцию; 1% за временной шаг. 
    			
			class BA2(sm.SM):
			    startState = 0 
			    def getNextValues(self, state, inp):
				newState = state * 1.01 + inp
				return (newState, newState)

Создайте автомат, который вычисляет баланс обоих типов, но выводит максимальный из двух.

Входными данными для автомата являются числа (положительные - это депозиты, отрицательные - это выплаты); вывод представляет максимум из двух балансов.

Начните с создания автомата, вход которого число, а выход tuple из двух балансов: баланс первого типа счета после ввода и баланс второго после ввода. Объедините эту машину с экземпляром pureFunction (определённой в предыдущем задании). Необходимые комбинаторы возьмите из модуля sm: sm.Cascade, sm.Parrallel, sm.Paralle2.

Шаги выполнения

Зайдите на cs50.dev. Сделайте рабочим каталогом вашу Codespace. Создайте каталог bankAccounts, выполнив 
mkdir bankAccounts
Перейдите в каталог. 
cd bankAccounts
Закачайте файлы sm.py, bankAccounts.py выполнив 

wget https://vadimgb.github.io/psets/data/sm.py 
wget https://vadimgb.github.io/psets/data/bankAccounts.py
Откройте файл bankAccounts.py, выполнив 

code bankAccounts.py
Дополните файл. Вы можете проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/bankAccounts local
Для отправки на проверку выполните команду 
npx cs204 2023/bankAccounts
