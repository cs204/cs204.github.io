Обучение модели классификации изображений с нуля

Шаги выполнения

Зайдите на cs50.dev. Закачайте архив 

wget https://cs204.github.io/psets/data/fashion.zip
Разархивируйте 

unzip fashion.zip
Перейдите в каталог 

cd fashion
Установите пакет 

sudo apt install graphviz
Создайте виртуальное окружение 

python -m venv .venv
source .venv/bin/activate
Установите нужные для проекта пакет 

pip install --upgrade pip
pip install -r requirements.txt
Слева в панели инструментов выберите Файл->Открыть папку. Выберите папку heart. Откройте файл example.ipynb, кликнув по нему мышкой два раза. В открытом окне выберите ядро python-jupyter. Виртуальная среда .venv Запустите проект на выполнение.

Создание собственного решения

Создайте файл 

touch fashion.ipynb
Откройте файл. Напишите собственный код на основе примера (example.ipynb). Разберите все команды. Вы можете проверить правильность выполнения задания, выполнив команду: 
npx cs204 2026/fashion local
Для отправки на проверку выполните команду: 
npx cs204 2026/fashion
Чтобы закрыть виртуальную среду, выполните команду: 
deactivate
