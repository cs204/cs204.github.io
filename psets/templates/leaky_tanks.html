Баки

На следующем рисунке показана каскадная система из двух резервуаров для воды. Вода течёт

  • в первый бак скорость \(r_0(t)\),
  • из первого бака и во второй скорость \(r_1(t)\),
  • скорость вытекания из второго \(r_2(t)\).

Скорость потока из каждого резервуара пропорциональна высоте воды в этом резервуаре: $$ \begin{split} r_1(t)&=k_1h_1(t),\\ r_2(t)&=k_2h_2(t),\\ \end{split} $$ где \(k_1\) и \(k_2\) каждый 0.2 метра2/секунду. Оба резервуара имеют высоту 1 м. Площадь поперечного сечения резервуара 1 \(A_1 = 4 м^2\), второго резервуара \(A_2 = 2 м^2\). В момент времени t = 0 оба резервуара пусты.

import math
import numpy as np
import matplotlib.pyplot as plt
#r1 rate of water flow out of tank [m^3/second]
#r0 rate of flow into the tank
#h - height of water
#A cross section of tank m^2
#k  [m^2/second] r=h*k
# A dh(t)/dt = r0(t) - r1(t)
# A/k dr1(t)/dt = r0(t)-r1(t)
# tau = A/k = 4*10/2=20 
A = 4 
k = 0.2 
tau = A/k
def y(t):
    return 1 - math.exp(-t/tau)

tMax = 200
t = np.arange(0, tMax, 0.2)
yr=[y(s) for s in t]


T = 1
n = int(tMax / T)

xd = [0]
for i in range(1, n+1):
    xd.append(i*T)



yd = [0]
for i in range(1, n+1):
    yd.append(yd[i-1]*(1 - T/tau) + T/tau)

plt.plot(t, yr, '-', xd, yd, 'ob')
plt.show()
