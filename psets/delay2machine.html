Двойная задержка

Создайте класс Delay2Machine автомата , который на два шага задерживает ввод. Вывод в момент i равен вводу в момент i-2. Конструктор класса берет два аргумента, они будут выводами в моменты 0, 1. 


sm = Delay2Mathine(100, 10)

Шаги выполнения

Зайдите на cs50.dev. Сделайте рабочим каталогом вашу Codespace. Создайте каталог delay2machine, выполнив 
mkdir delay2machine
Перейдите в каталог. 
cd delay2machine
Закачайте файлы sm.py, delay2machine.py выполнив 

wget https://vadimgb.github.io/psets/data/sm.py 
wget https://vadimgb.github.io/psets/data/delay2machine.py
Откройте файл delay2machine.py, выполнив 

code delay2machine.py
Дополните файл.

Как проверить

  • Сделайте рабочим каталог с программой,
  • Запустите программу, выполнив python delay2machine.py.
Вы можете проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/delay2machine local
Для отправки на проверку выполните команду 
npx cs204 2023/delay2machine
