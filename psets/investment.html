Инвестиции

Рассмотрим два разных типа банковских счетов:

  • B1: комиссия в размере 100$ за каждый (ненулевой) депозит и вывод средств; 2% годовых за каждый временной шаг. 
    			
			class BA1(sm.SM):
			    startState = 0
			    def getNextValues(self, state, inp):
				if inp != 0:
				    newState = state * 1.02 + inp - 100
				else:
				    newState = state * 1.02
				return (newState, newState)
  • B2: отсутствие комиссии за транзакцию; 1% за временной шаг. 
    			
			class BA2(sm.SM):
			    startState = 0 
			    def getNextValues(self, state, inp):
				newState = state * 1.01 + inp
				return (newState, newState)

У бизнесмена есть два счёта, описанных выше. Вклады и снятия по величине выше $3000 он делает на счёте первого типа, другие на счёте второго типа. На каждом шаге, оба счёта должны пополняться по процентрой ставке. Вывод должен быть суммой балансов обоих счетов.

Реализуйте это. Объедините оба счета используя sm.Parallel2, полученную машину соедините каскадно с PureFunction. Присвойте вашей машине имя switchAccount. Воспользуйтесь файлами приведёнными ниже.

Шаги выполнения

Зайдите на cs50.dev. Сделайте рабочим каталогом вашу Codespace. Создайте каталог investment, выполнив 
mkdir investment
Перейдите в каталог. 
cd investment
Закачайте файлы sm.py, investment.py выполнив 

wget https://vadimgb.github.io/psets/data/sm.py 
wget https://vadimgb.github.io/psets/data/investment.py
Откройте файл investment.py, выполнив 

code investment.py
Дополните файл. Вы можете проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/investment local
