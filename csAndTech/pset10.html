Начало

Здесь мы рассмотрим как загрузить задание в вашу CS50 IDE. Зайдите в CS50 IDE, затем в терминальном окне выполните каждую из команд, приведённых ниже.

  • Выполните cd ~/, чтобы перейти в домашний каталог.
  • Выполните mkdir pset10, чтобы создать каталог pset10 в вашем домашнем каталоге.
  • Выполните cd pset10, чтобы сделать pset10 рабочей директорией.
  • Выполните wget https://vadimgb.github.io/data/houses.zip, чтобы загрузить архив с дистрибутивом задания.
  • Выполните unzip houses.zip, чтобы распаковать архив.
  • Выполните rm houses.zip и y, чтобы удалить архив.
  • Выполните приведённых. Вы должны увидеть директорию houses, которая была в файле архиве.
  • Выполните cd houses, чтобы перейти в каталог.
  • Выполните ls. Вы должны увидеть characters.csv файл и students.db файл базы данных.

Основание

Хогвартс нуждается в базе данных студентов. В течение многих лет профессора в CSV файле сохраняли имена студентов, их факультеты и годы рождения. Но с помощью такого файла трудно выбрать нужные данные, такие как список всех студентов факультета Пуффендуй или алфавитный список всех зачисленных в школу.

Ваша задача - импортировать все школьные данные в SQLite базу данных и написать Python программу, запрашивающую эту базу данных выдать список студентов заданного факультета.

Детализация

В import.py напишите программу, импортирующую данные из таблицы CSV.

  • Ваша программа должна принимать имя CSV файла как аргумент командной строки.
    • Если число аргументов введённой командой строки неверно, то программы должна сообщить об ошибке и завершить работу.
    • Файл character.csv, содержит столбцы имя, факультет, рождение.
  • Для каждого студента в CSV файле, создайте запись в таблице студенты в students.db базе данных.
    • В то время как csv файл просто столбец имя студента, таблица студенты базы данных должна содержать отдельные столбцы имя, отчество, фамилия. Сначала вы должны разделить имя на имя, фамилию и отчество. В файле поле имя содержит или два разделённых пробелом имени (имя, фамилия) или три (имя, отчество и фамилия). Для студентов без отчества вы должны поле отчество сделать NULL в таблице.

В roster.py напишите программу, которая печатает список студентов данного факультета в алфавитном порядке.

  • Ваша программа должна принимать имя факультета как аргумент командной строки.
    • Если введено неверное количество аргументов командной строки, программа сообщает об ошибке и завершает работать.
  • Ваша программа должна запрашивать таблицу студенты в students.db базе данных о всех студентах факультета.
  • Затем ваша программ должна распечатать для каждого студента его полное имя и год рождения (форматированное как, например, Гарри Джеймс Поттер, родился 1980).
    • Каждый студент печатается на отдельной строке.
    • Студенты упорядоченны по фамилии. Студенты с одинаковой фамилией упорядочены по имени.

Рекомендации

  • После импортирования SQL из cs50, вы можете установить соединение с базой данных 
    		    
		    db = cs50.SQL("sqlite:///students.db")

Затем вы можете использовать db.execute, чтобы выполнить sql запрос в Python программе.

  • Напомним, что когда вы вызываете db.execute с запросом SELECT, то результирующее значение будет список строк, каждая строка представлена словарём Python.
