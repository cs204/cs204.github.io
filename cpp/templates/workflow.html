Open source workflow.

Зайдите на https://github.com/cs203/calc. Нажмите на кнопку Fork. У вас будет создана копия репозитория. Скопируйте его адрес. Зайдите в cs50.dev. Отключите плагин gitdoc. Перейдите в домашний каталог, сколонируйте репозиторий. 


cd ~
git clone адрес
Перейдите в каталог calc. Создайте ветку dev2, перейдите в неё. 

cd calc 
git branch dev2
git checkout dev2
Добавьте свою фамилию в readme.md 

git add .
git commit -m "ваше название"
Добавьте информацию об основном репозитории. Перейдите в ветку master. 

git remote add upstream https://github.com/cs203/calc
git checkout master
Закачайте изменения. Соедините их с вашими. Загрузите результат. 

git fetch upstream
git merge upstream
git merge dev2
git push origin master
Зайдите на github, создайте pull request.
