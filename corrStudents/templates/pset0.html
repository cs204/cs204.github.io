Регистрация на сайте учета заданий

Выполнив задание из лекции, вы зарегистрировались на github. Используя учётную запись на github, зайдите на сайт https://cs204.herokuapp.com. Вам будет отправлено письмо с темой cs203. Примите находящееся в нём приглашение (View Invitation -> Accept Invitation). Письмо может попасть в спам. Для вас был создан репозиторий, вы будете отправлять в него свой работы на проверку. Имя репозитория имеет вид https://github.com/cs204/{username}.git, где {username} заменяется на имя пользователя. Так если пользователь имеет имя в github vadimb1, то репозиторий будет называться https://github.com/cs204/vadimgb1.git, где cs204 - это имя организации, а vadimgb1 имя на github.

Выполнение первого задания

Зайдите на ide.cs50.io. Перейдите в домашний каталог 
$ cd ~
Создайте клон репозитория 
$git clone https://github.com/cs204/{username}.git -b main pset2
В адресе замените {username} на ваше имя на github. Перейдите в папку pset1 
cd pset2
Укажите имя и адрес почты, которые будут использоваться глобально для ваших коммитов 
git config --global user.name "Ваше имя"
git config --global user.email "ваш email"
Создайте новую ветку и перейдите в неё 
git checkout -b pset2
Создайте файл hello.js и сохраните его в папке pset2. Текст файла ниже 

console.log('Hello, world!')
Выполните команду в терминале 
node hello.js
Интерпретатор node выполнил нашу программу и вывел на экран приветствие. Чтобы проверить правильность выполнения задания, выполните команду 
check50 cs204/psets/2021/hello --local
Сохраните изменения в репозитории и отправьте их на github. 
git add hello.js
git commit -m "first commit"
git push origin -f pset2
