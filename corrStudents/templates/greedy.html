Задача о рюкзаке

Жадный алгоритм

Используя жадный алгоритм с сортировкой по удельной цене, решите задачу о рюкзаке из лекции.

Шаги выполнения

Зайдите на ide.cs50.io. Перейдите в домашний каталог 
cd ~
Создайте клон репозитория 
git clone https://github.com/cs204/{username}.git -b main pset4
В адресе замените {username} на ваше имя на github. Перейдите в папку pset4 
cd pset4
Создайте новую ветку и перейдите в неё 
git checkout -b pset4
Закачайте в папку файл greedy.js 
 wget https://vadimgb.github.io/weeks/greedy.js
В этом файле вам надо определить функцию сравнения по удельной цене cmpDencity. Подробно об этом рассказано в видео к лекции. Выполните программу 
 node greedy.js
Интерпретатор node выполнил нашу программу. Чтобы проверить правильность выполнения задания, выполните команду 
check50 cs204/psets/2021/greedy --local -o html
Сохраните изменения в репозитории и отправьте их на github. 
git add greedy.js
git commit -m "first commit"
git push origin -f pset4
