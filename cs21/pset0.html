Регистрация на сайте учета заданий

Выполнив задание из лекции, вы зарегистрировались на github. Используя учётную запись на github, зайдите на сайт http://90.188.117.161:8080. Вам будет отправлено письмо с темой cs203. Примите находящееся в нём приглашение (View Invitation -> Accept Invitation). Письмо может попасть в спам. Для вас был создан репозиторий, вы будете отправлять в него свой работы на проверку. Имя репозитория имеет вид https://github.com/cs204/{username}.git, где {username} замените на имя пользователя на github. Так если пользователь имеет имя в github vadimgb1, то репозиторий будет называться https://github.com/cs204/vadimgb1.git, где cs204 - это имя организации.

Выполнение первого задания

Зайдите на code.cs50.io. Создайте каталог и перейдите в него. 
mkdir ~/hello
cd ~/hello
Создайте файл hello.js , сохраните его в папке ~/hello. Текст файла ниже 

console.log('Hello, World!')
Выполните команду в терминале 
node hello.js
Интерпретатор node выполнил вашу программу и вывел на экран приветствие. Чтобы проверить правильность выполнения задания, выполните команду 
npx cs204 2021/hello local
Для отправки на проверку выполните команду 
npx cs204 2021/hello
