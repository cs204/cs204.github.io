Express

Создание веб-приложения.

Создайте директорию express. Перейдите в неё. Выполните команду: 


npm init -y
Эта команда создаст package.json файл, который содержит описание вашего приложения.

Добавьте в файл package.json строку 


"type":"module"

В этом файл используется формат json, в нём в фигурных скобках, через запятую перечислены пары атрибут-значения, атрибут отделяется от значения двоеточием. Установите пакеты express, ejs, body-parser, выполнив 


npm install express ejs body-parser

Создайте index.js файл.

Создайте каталоги views, public. В парку public поместите файл, эмблему тгпу. Создайте index.ejs файл в папке views.

Создайте hello.ejs файл в папке views.

Запустите ваше приложение выполнив:


node index.js

Выложите веб-приложение на https://render.com

Инициализируйте git в каталоге вашего приложения, выполнив


git init

Создайте файл .gitinit . Добавьте в него строки


node_modules
.env

Эти строки нужны, чтобы git не создавал копии указанных файлов. Создайте репозиторий на github. Загрузите в него ваше приложение. Зарегистрируйтесь на render.com. Разместите на нем своё приложение.
