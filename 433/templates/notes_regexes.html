Регулярные выражения

Регуля́рные выраже́ния (англ. regular expressions) — формальный язык, используемый для поиска и осуществления манипуляций с подстроками в тексте, основанный на использовании метасимволов. Для поиска используется строка-образец (англ. pattern), состоящая из символов и метасимволов и задающая правило поиска.

Для манипуляций с текстом дополнительно задаётся строка замены, которая также может содержать в себе специальные символы.

python имеет библиотеку re в ней есть функции использующие регулярные выражения. Справка по re.

Рассмотрим примеры использования этой библиотеки. Например, нам может потребоваться проверить правильность формата адреса электронной почты. Создадим файл validate.py. Для начала введите code validate.py в окне терминала. Затем напишите в текстовом редакторе следующий код: 


import re
email = input("What's your email? ").strip()

if re.search(".+@.+", email): 
    print("Valid")
else: 
    print("Invalid")
Ниже приведены некоторые специальные символы языка регулярных выражений. 

. любой символ, кроме новой строки
* 0 или больше повторений
+ 1 или больше повторений
? 0 или 1 потвторение
{m} m повторений
{m, n} m-n повторений
^ обозначает начало строки
$ соответствует концу строки
[] множество симолов
[^] дополнение к множетству
\d десятичные цифры
\D не десятичные цифры
\s пробельные символы
\S не пробельные символы
\w буквы, цифры, символ подчёркивания (word character)
\W not word character 
A|B A или  B 
(...) группа
(?:...) группа, но без захвата

Теперь давайте извлечем некоторую конкретную информацию из пользовательского ввода. В окне терминала введите code twitter.py и введите следующий код в окне текстового редактора: 


import re
url = input("URL: ").strip()
matches = re.search(r"^(?:https?://)?(?:www\.)?twitter\.com/(\w+)$", url, re.IGNORECASE)
if matches:
    print(f"Username:", matches.group(1))
