Банк.

В 7 сезоне, эпизод 24 сериала Сайнфелд, Крамер заходит в банк, который обещал дать $100 тому, кого не встретят с "здравствуйте". Крамера встретили с "здрасте", а не "здравствуйте", и он потребовал $100. Менеджер банка предложил компромис:"Вы получили приветствие начиналось оно с "з", $20?" Крамер принял предложение.

В файле bank.py, реализуйте программу, запрашивающую у пользователя приветствие(input("Приветствие: ")). Если приветствие начинается с "здравствуйте", то вывод $0. Если приветствие начинается с "з"(но не здравствуйте), вывод $20. В других случаях вывод $100.

Подсказка
  • Тип str имеет много методов, см https://digitology.tech/docs/python_3/library/stdtypes.html#string-methods
  • $0 выводим, не только для "здравствуйте", но и для "здравствуйте Крамер", и тому подобное.

Демонстрация

Шаги выполнения

Зайдите на code.cs50.io. Сделайте рабочим каталогом вашу Codespace. Создайте каталог bank, выполнив 
mkdir bank
Перейдите в каталог. 
cd bank
Выполните 

code bank.py
чтобы создать файл bank.py, в котором вы будете писать вашу программу.

Как проверить

  • Сделайте рабочим каталог с программой,
  • Запустите программу, выполнив python bank.py. Введите здравствуйте и нажмите Enter. Программа должна вывести: 
    	
	$0
  • Запустите программу, выполнив python bank.py. Введите здравствуйте, человек и нажмите Enter. Программа должна вывести: 
    	
	$0
  • Запустите программу, выполнив python bank.py. Введите здрасте и нажмите Enter. Программа должна вывести: 
    	
	$20
  • Запустите программу, выполнив python bank.py. Введите привет и нажмите Enter. Программа должна вывести: 
    	
	$100
Вы можеет проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/bank local
Для отправки на проверку выполните команду 
npx cs204 2023/bank
