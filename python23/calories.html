Калорийность

Посмотрите таблицу пищевой ценности для 20 популярных фруктов.

В файле calories.py реализуйте программу, приглашающую пользователя ввести фрукт (input("Фрукт: ")) и возвращающую число калорий на порцию (например, print(f"Калории: {calories}")). Предположим, что пользователь будет вводить названия точно, как они записаны в таблице.

Используйте словарь для связывания фруктов с калориями. Если k типа str и d типа dict, вы можете проверить есть ли k в d: 


if k in d:
	....

Демонстрация

Шаги выполнения

Зайдите на code.cs50.io. Используя команду cd, сделайте рабочим каталогом codespace. Создайте каталог camel, выполнив 
mkdir calories
Перейдите в каталог. 
cd calories
Выполните 

code calories.py
чтобы открыть редактор, и напишите вашу программу.

Как проверить

  • Сделайте рабочим каталог с программой.
  • Запустите программу python calories.py. Введите Apple и нажмите Enter. Ваша программа должна вывести 
    
Калории: 130
  • Запустите программу python calories.py. Введите Avocado и нажмите Enter. Программа должна вывести Программа должна вернуть 
    
Калории: 50
  • Запустите программу python calories.py. Введите Tomato и нажмите Enter. Программа не должна ни чего вывести.
Вы можеет проверить правильность выполнения задания, выполнив команду: 
npx cs204 2023/calories local
Для отправки на проверку выполните команду: 
npx cs204 2023/calories
