code.cs50.io

Регистрация и вход в среду разработки.

Чтобы не устанавливать на домашний компьютер программное обеспечение, необходимое для выполнения заданий, мы будем пользоваться средой на облаке code.cs50.io. Входить в эту среду мы будем через учётную запись на github.com.


Команды терминала

  • ls - список файлов и папок
  • mkdir - создать каталог
  • rm - удалить файл или каталог
  • cd - изменить рабочий каталог
  • pwd - вывести рабочий каталог
  • cp - копировать файл или папку
  • mv - переместить файл или папку
  • man - вывести справку
  • code - открыть редактор
Через терминал мы будем общаться с ядром операционной системы. Ядро операционной системы выполняет две функции. Первая функция - предоставление программам доступа к оборудованию, вторая - многозадачность. Используя язык sh в терминале мы будем создавать папки, запускать программы... Создадим паку с именем week0 в домашнем каталоге пользователя. Для этого выполните в окне терминала команду


~/$ mkdir ~/week0

~ - короткое обозначение домашнего каталога. Текущий каталог, или рабочий каталог - это каталог, в котором программа начинает поиск файлов, если не указан полный путь. Следующая команда сделает текущим каталогом каталок week0. 


~/$ cd ~/week0
