Назад в банк.

В файле bank.py, реализуйте программу из задания банк, изменив её, как показано ниже, где функция value ожидает строку приветствия и если приветствие начинается с "здравствуйте", то возвращает чилсо 0, если приветствие начинается с "з"(но не здравствуйте), возращает 20. В других случаях возвращает 100. Печать осуществляет main. 


def main():
	...

def value(greeting):
	....

if __name__ == "__main__":
	main()
В файле test_bank.py реализуйте функции тестирующие value. Запустите pytest test_bank.py.
Подсказка
  • Не забудьте в начало файла включить 
    
import bank
    или 
    
from bank imoprt value
  • value должна использовать return, а не print. print вызывает main.

Шаги выполнения

Зайдите на code.cs50.io. Сделайте рабочим каталогом вашу Codespace. Создайте каталог test_bank, выполнив 
mkdir test_bank
Перейдите в каталог. 
cd test_bank
Создайте файл bank.py, реализуйте программу. Выполните 

code test_bank.py
чтобы создать файл test_bank.py, в котором вы будете писать тест.

Как проверить

Сделайте рабочим каталог с программой, Запустите программу, выполнив pytest test_bank.py. Попробуйте правильные и неправильные версии файла bank.py, чтобы посмотреть как работают ваши тесты:
  • Убедитесь, что файл bank.py корректный. Запустите тест, выполнив pytest test_bank.py. pytest должен показать, что программа прошла тест без ошибок.
  • Измените bank.py, поменяв значения, возвращаемые на приветствия, например, чтобы программа выдавала $100, если приветствие начинается на "здравствуйте". Запустите, выполнив pytest test_bank.py. pytest должен показать ошибки.
Вы можете проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/test_bank local
Для отправки на проверку выполните команду 
npx cs204 2023/test_bank
