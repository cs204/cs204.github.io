В файле deep.py реализуйте программу на Pyhton, которая приглашает пользователя ответить на главный вопрос жизни, вселенной и всего такого, выводит Да, если пользователь отвечает 42 или сорок два (без учёта регистра), иначе выводит Нет.

Подсказка

Демонстрация

Шаги выполнения

Зайдите на code.cs50.io. Выполните команду: 
cd 


Создайте каталог deep,  выполнив

mkdir deep


Перейдите в каталог.

cd deep


Выполните 


code deep.py



 и напишите вашу программу.

Как проверить


Сделайте рабочим каталог с программой, запустите программу, выполнив:


python deep.py




Введите 42 и нажмите Enter. Ваша программа должна вывести: Да.
Запустите вашу программу, введите сорок два и нажмите Enter.
Программа должна вывести: Да. Запустите программу и введите 32,
программа должна вывести: Нет.


Вы можеет проверить правильность выполнения задания, выполнив команду

npx check204 2023/deep local


Для отправки на проверку выполните команду

npx check204 2023/deep
