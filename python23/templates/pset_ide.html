Регистрация на сайте учета заданий

Выполнив задание из лекции, вы зарегистрировались на github, создали токен. Используя учётную запись на github, зайдите на сайт http://90.188.117.161:8080. Вам будет отправлено письмо с темой cs203. В веб-браузере устройства, на котором будете смотреть письмо, надо войти на github. Примите находящееся в письме приглашение (View Invitation -> Accept Invitation). Письмо может попасть в спам. Для вас был создан репозиторий, вы будете отправлять в него свой работы на проверку. Имя репозитория имеет вид https://github.com/cs204/{username}.git, где {username} замените на имя пользователя на github. Так если пользователь имеет имя в github vadimgb1, то репозиторий будет называться https://github.com/cs204/vadimgb1.git, где cs204 - это имя организации.

Выполнение первого задания

Зайдите на ide.cs50.io. Создайте каталог и перейдите в него. 
mkdir ~/hello
cd ~/hello
Создайте файл hello.py, сохраните его в папке ~/hello. Текст файла ниже 

print('Hello, World!')
Выполните команду в терминале 
python hello.py
Интерпретатор python выполнил вашу программу и вывел на экран приветствие. Чтобы проверить правильность выполнения задания, выполните команду 
npx check204 2023/hello_py local
Для отправки на проверку выполните команду 
npx check204 2023/hello_py
