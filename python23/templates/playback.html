Скорость воспроизведения

В файле playback.py реализуйте программу, которая считывает ввод пользователя и возвращает его, заменив все пробелы на ...(на три точки).

Подсказка
  • Напомним, что input возвращает тип str, см. docs.python.org/3/library/functions.html#input
  • Тип str имеет много методов, см docs.python.org/3/library/stdtypes.html#string-methods

Шаги выполнения

Зайдите на ide.cs50.io. Выполните команду: 
cd 


Создайте каталог playback,  выполнив

mkdir playback


Перейдите в каталог.

cd playback


Выполните 


code playback.py



чтобы открыть редактор, и напишите вашу программу.

Как проверить


Сделайте рабочим каталог с программой, запустите программу, выполнив:


python indoor.py



Введите мама мыла раму и нажмите Enter. Ваша программа должна вывести мама...мыла...раму.

Вы можеет проверить правильность выполнения задания, выполнив команду

npx check204 2023/playback local


Для отправки на проверку выполните команду

npx check204 2023/playback
