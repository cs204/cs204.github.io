Кофе

Аппарат продаёт кофе за 50 рублей стакан и принимает только монеты по 50, 20, 10, 5 рублей.

В файле coffee.py реализуйте программу, она приглашает пользователя вставить монету (coin = input("Вставьте монету: ")), каждый раз информирую пользователя о нужной сумме (например, print(f"Нужная сумма: {amount_due}")). Как только пользователь ввёл не менее 50 рублей, программа выводит сумму сдачи (например, print(f"Ваша сдача: {change_owed}")).

Демонстрация

Шаги выполнения

  • Зайдите на code.cs50.io. Используя команду cd, сделайте рабочим каталогом codespace. Создайте каталог coffee, выполнив 
    mkdir coffee
    Перейдите в каталог. 
    cd coffee
  • Откройте редактор 
    
code coffee.py
    Создайте вашу программу.

    Как проверить

    Сделайте рабочим каталог с программой. Запустите программу 

    
python coffee.py
