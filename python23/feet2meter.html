Но не успели задать функцию feet2meter. Определите эту функцию. Она должна принимать аргумент str (вида ##.##ft , где каждый # это десятичная цифра), удалять ft и возвращать результат типа float. Например, дано 180.9ft как ввод, она должна вернуть 55.138.

Подсказка
  • Напомним, что input возвращает тип str, см. https://digitology.tech/docs/python_3/library/functions.html#input
  • Напомним, что float может конвертировать str в float, docs.python.org/3/library/functions.html#float
  • Тип str имеет много методов, см https://digitology.tech/docs/python_3/library/stdtypes.html#string-methods

Демонстрация

Шаги выполнения

Зайдите на code.cs50.io. Выполните команду: 
cd 


Создайте каталог feet2meter,  выполнив

mkdir feet2meter


Перейдите в каталог.

cd feet2meter


Выполните 


code feet2meter.py



чтобы открыть редактор. Скопируйте и вставьте код выше и завершите 
реализацию функции feet2meter.


Как проверить


Сделайте рабочим каталог с программой, запустите программу, выполнив:


python feet2meter.py



Введите 180.9ft и нажмите Enter. Ваша программа должна вывести:


Это будет 55.14 метров.




Вы можеет проверить правильность выполнения задания, выполнив команду

npx cs204 2023/feet2meter local


Для отправки на проверку выполните команду

npx cs204 2023/feet2meter
