СтильВерблюда

СтильВерблюда стиль написания составных слов, при котором несколько слов пишутся слитно без пробелов, при этом каждое слово внутри фразы пишется с прописной буквы. Стиль получил название стильВерблюда, поскольку прописные буквы внутри слова напоминают горбы верблюда. Например, переменную для имени пользователя можно назвать firstName.

Змеиный_стиль - стиль написания составных слов, при котором несколько слов разделяются символом подчёркивания (_), и не имеют пробелов в записи, причём каждое слово обычно пишется с маленькой буквы — first_name и т. д.

В файле camel.py реализуйте программу, приглашающую ввести имя переменной в верблюжьемСтиле ( input("Верблюжий стиль: ")), и возвращающую имя в змеином_стиле.

Подсказка
  • Тип str имеет много методов, см https://digitology.tech/docs/python_3/library/stdtypes.html#string-methods
  • Как и список, строку можно обрабатывать в цикле. Например, если s строка, то мы можем распечатать каждую букву отдельно: 
    			
			for c in s:
				print(c, end="")

Демонстрация

Шаги выполнения

Зайдите на code.cs50.io. Используя команду cd, сделайте рабочим каталогом codespace. Создайте каталог camel, выполнив 
mkdir camel
Перейдите в каталог. 
cd camel
Выполните 

code camel.py
чтобы открыть редактор, и напишите вашу программу.

Как проверить

Сделайте рабочим каталог с программой, запустите программу, выполнив: 


python camel.py
Введите firstName и нажмите Enter. Ваша программа должна вывести 

firs_name
Введите prefeeredFirstName и нажмите Enter. Программа должна вернуть 

prefeered_first_name
Вы можеет проверить правильность выполнения задания, выполнив команду 
npx cs204 2023/camel local
Для отправки на проверку выполните команду 
npx cs204 2023/camel
